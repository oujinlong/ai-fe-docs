##模版支持
1.[vue-template](./vue-template.html) (vue项目模版)

2.React Native (持续更新..)

3.微信小程序wechat app (持续更新..)

4.网页公众号 (持续更新..)

##模板安装
###环境要求
* node.js 版本 >= 8.0
* npm 版本 >= 5.0
* vue 版本 >= 2.0 (vue 模版所需)

###安装步骤
	全局安装脚手架
	npm install -g ai-fe-cli
	
	创建项目模版
	ai-fe init
	
	安装完成
